# 区块备注器 (BlockNote)

一个基于 AI 的区块备注工具——纯 HTML + JavaScript 实现，支持可编辑网格、备注标注与本地存储。

## 项目简介

区块备注器是一款轻量级网页应用，用于标记、编辑和管理无限网格中的区块备注。  
它基于纯 HTML/CSS/JS 构建，**无需后端**，所有数据保存于浏览器的本地存储（localStorage）中。

> 🤖 **AI 集成说明**  
> 该项目预留了与 DeepSeek AI 的结合能力，可用于：
> - 自动生成区块备注摘要
> - 根据内容智能推荐标签
> - 语义理解与智能检索

## 核心功能

- ✅ **无限网格坐标系**：以 (X, Y) 定位区块，支持任意范围  
- ✅ **拖拽 & 缩放**：鼠标/触摸拖动平移，滚轮缩放网格  
- ✅ **区块备注**：选中任意格子，添加/编辑/删除文字备注  
- ✅ **视觉标记**：被备注过的区块带有 📌 图标与高亮  
- ✅ **跳转定位**：快速跳转到指定坐标区块  
- ✅ **持久化存储**：备注数据自动保存在浏览器中  
- ✅ **全响应式**：支持 PC / 平板 / 手机触摸操作  

## 技术栈

| 类型       | 技术                           |
|-----------|-------------------------------|
| 前端      | 原生 HTML5 + CSS3 + JavaScript |
| 图形渲染  | Canvas 2D API                 |
| 存储      | localStorage (键值JSON存储)    |
| 交互      | 鼠标/触摸事件 + 滚轮缩放        |
| 可扩展    | 预留 DeepSeek API 集成接口     |
